
# PT: All Pro

Apps multitude lead to clutter on the phone, how about one full-featured app that offers you:

1️⃣ Alarm Clock ⏰⏰  
2️⃣ Gallery 🌄🎞️  
3️⃣ File manager 📁📂  
4️⃣ Music player 🎧🎼  
5️⃣ Notepad with Counter 🗒️🗓️  
6️⃣ Screen Recorder app and screenshot
